# Pantalla 7 — Configuración / Settings

## Secciones

- **Cuenta** — email, contraseña, eliminación de cuenta.
- **Notificaciones** — toggle por tipo:
  - Streamers seguidos
  - Solicitudes
  - Eventos
  - Resumen semanal
- **Privacidad** — zona horaria, idioma, ubicación.
- **Wallet** — saldo, métodos de pago, historial, retirar.
- **Para streamers** — configuración de pagos (Stripe Connect), KYC.
- **Ayuda y soporte**.
- **Términos y privacidad**.

## Navegación inferior (5 íconos)

1. **Explorar** (home).
2. **Misiones** (solicitudes).
3. **LIVE** — botón circular grande, rojo, sobresale 20 px arriba del nav bar.
4. **Ver** — último stream visto, o lista de seguidos.
5. **Perfil**.

## Notas

- La frecuencia de notificaciones está **regulada por reglas globales** que el usuario no puede romper hacia arriba (máximo 2/día). Ver [[Notificaciones Push]].
- Las propinas **no se pueden desactivar**, ni siquiera desde esta pantalla. Ver [[Reglas de UX que no se negocian]].
