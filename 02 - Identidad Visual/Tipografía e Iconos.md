# Tipografía e Iconos

## Tipografía

### Principal
Elegir una de las tres: **Inter**, **Outfit** o **Space Grotesk**. Todas son sans-serif modernas.

### Para datos / IDs
**JetBrains Mono** o **IBM Plex Mono**.

### Pesos disponibles
- 400 (regular)
- 500 (medium)
- 600 (semibold)
- 700 (bold)
- 800 (extrabold)

### Tamaños

| Rol | Tamaño | Peso |
|---|---|---|
| Hero | 48-56 px | 800 |
| Título grande | 32-40 px | 700 |
| Título | 24-28 px | 700 |
| Subtítulo | 18-20 px | 600 |
| Cuerpo | 15-16 px | 400 |
| Pequeño | 13-14 px | 400 |
| Micro | 11-12 px | 500 (etiquetas, badges) |

## Iconografía

- **Estilo:** outline / stroke, **no rellenos**.
- **Grosor:** 2 px.
- **Esquinas:** redondeadas.
- **Librería sugerida:** Lucide React (lucide.dev) — gratuita, completa, consistente.

## Espaciado, bordes y sombras

- **Espaciado:** sistema 4 px (4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 80, 96).
- **Radios:** 8 px (chico), 12 px (medio), 16 px (grande), 22 px (XL), 999 px (pill).
- **Sombras:** sutil, media, fuerte, y un **glow rojo** específico para botones live: `0 4px 20px rgba(255,59,78,0.3)`.
- **Animaciones:** 200-300 ms con `ease-out`. Hover en cards: `translateY(-2px)`. Punto rojo del logo: pulse continuo.
