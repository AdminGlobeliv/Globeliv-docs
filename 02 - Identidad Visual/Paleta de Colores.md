# Paleta de Colores

## Colores primarios

| Nombre | Hex | Significado |
|---|---|---|
| Azul GlobeLiv | `#3B82F6` | Planeta, tecnología, confianza |
| Rojo Live | `#FF3B4E` | Transmisión, energía, urgencia |

## Fondos y texto (modo oscuro)

| Nombre | Hex | Uso |
|---|---|---|
| Negro / Oscuro | `#0A0E1A` | Fondo principal |
| Fondo card | `#141828` | Cards y elementos |
| Texto principal | `#F1F5F9` | Sobre fondos oscuros |
| Texto secundario | `#94A3B8` | Subtítulos, metadatos |
| Texto muted | `#64748B` | Timestamps, etiquetas menores |

## Colores semánticos

| Nombre | Hex | Significa |
|---|---|---|
| Verde | `#22C55E` | Ganancias, dinero, éxito |
| Dorado | `#F59E0B` | Precios, recompensas, premium |
| Borde sutil | `rgba(255,255,255,0.08)` | Separadores en oscuro |

## Notas

- La paleta está pensada **modo oscuro primero**.
- El rojo `#FF3B4E` se reserva para "live" y acciones primarias críticas (botón Go Live).
